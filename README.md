# Customer Segmentation
This project analyzes customers behavior to identify different customer segment base on their income, generation and spending patterns.  - The goal is to help businesses understand their customer and design better marketing strategies.
# Tools used 
- Microsoft Excel
- Power query
- SQL Sever

 # Database Description 
 The datase contains customers demographic and spreading information 

#### column includes:
 - custormer ID
 - Gender
 - Age
 - Annual Income
 - Spending score

 ####  SQL Analysis
   Sql Queries were used to perform exploratory analysis. including
   - Total Customers
   - Average income
   - Spending behaviour by gender
   - Customer Segmentation

#### Customer Segmentation
     Customer were segemented into four groups
     High value     -      High Income and high Spending
     Careful        -      high income but low spending
     potential      -      Low income but i spending
     low Value      -      low income and low spending

  - High income customers tends to have higher spending scores.
